# Study Habits vs Academic Performance Analysis

## 📌 Overview
This project explores the relationship between student study habits and academic performance using data analysis and visualization. The goal is to identify patterns between factors such as study time, sleep, and perceived effectiveness, and how they influence grades.

---

## 🎯 Objectives
- Analyze how study habits impact academic performance  
- Visualize relationships between key variables  
- Explore behavioral patterns in student data  
- Build a simple predictive model based on observed trends  

---

## 🛠️ Technologies Used
- HTML  
- JavaScript  
- Chart.js  

---

## 📂 Dataset
The dataset includes:
- Study hours per day  
- Sleep hours per night  
- Screen time  
- Class attendance  
- Grade range (converted to numeric values)  
- Self-reported study effectiveness  

The data consists of a combination of:
- Survey responses collected from students  
- Simulated data to support analysis  

---

## Features

### 1. Data Visualization
Scatter plots were created to analyze:
- Study Hours vs Grades  
- Sleep vs Grades  
- Study Effectiveness vs Grades  

---

### 2. Insights
Key observations from the analysis include:
- Higher study time generally correlates with higher grades  
- Increased sleep is associated with better academic performance  
- Students who report higher study effectiveness tend to achieve higher grades  

---

### 3. Prediction Model
A simple rule-based model was implemented to estimate expected grade outcomes based on:
- Study hours  
- Sleep duration  
- Study effectiveness  

This demonstrates how basic decision rules can approximate patterns in real-world data.

---

## Limitations
- Small sample size may affect accuracy  
- Some data was simulated  
- Responses are self-reported and may include bias  

---

## 🚀 Future Improvements

This project demonstrates how data analysis and simple computational models can be used to explore human behavior and decision-making. It highlights the role of data-driven insights in understanding and improving real-world outcomes.
